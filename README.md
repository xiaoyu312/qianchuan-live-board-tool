# 千川直播大屏采集工具

这是本地桌面版工具，内置浏览器打开千川页面，登录态会保留。工具会按选择的时间间隔采集直播大屏数据，先展示在实时表格里，确认后再导出中文 CSV。

## 使用方式

1. 打开工具后，在右侧浏览器登录千川。
2. 打开直播大屏页面。
3. 选择采集间隔：`5分钟`、`15分钟`、`30分钟`、`1小时`。
4. 填写下播时间。
5. 在主播排班里填写主播时段，例如 `08:00-12:00 张三`、`12:00-15:00 李四`。
6. 点击开始采集，数据会先进入实时登记表。
7. 确认数据后点击导出表格并选择保存位置。

## 输出文件

点击导出后，工具会在你选择的文件夹里生成：

- `2026-06-30_直播大屏累计数据.csv`
- `2026-06-30_直播大屏时段数据.csv`
- `2026-06-30_直播大屏采集日志.jsonl`

CSV 都带中文表头，可直接用 Excel 打开。表格会包含 `主播`、`商品点击率`、`商品转化率`，主播按你填写的排班自动匹配。

## 自动停止规则

到下播时间后，工具会继续在下一个采集点检查一次。如果这一段时间内累计数据没有变化，就自动停止，并且不写入重复行。

## 本地运行

```bash
npm install
npm run dev
```

## 打包 Windows 安装包

```bash
npm run pack:win
```

打包产物在 `release/` 文件夹。
