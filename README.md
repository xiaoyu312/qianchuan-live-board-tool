# 千川直播大屏采集工具

这是本地桌面版原型，内置浏览器打开千川页面，登录态会保留。工具每 5 分钟整点采集一次直播大屏数据，并输出中文 CSV 表格。

## 使用方式

1. 打开工具后，在右侧浏览器登录千川。
2. 打开直播大屏页面。
3. 在左侧填写直播开始和结束时间，例如 `08:00` 到 `15:00`。
4. 在主播排班里填写主播时段，例如 `08:00-12:00 张三`、`12:00-15:00 李四`。
5. 选择表格保存位置。
6. 点击开始采集。

## 输出文件

工具会在你选择的文件夹里生成：

- `2026-06-30_直播大屏累计数据.csv`
- `2026-06-30_直播大屏5分钟时段数据.csv`
- `2026-06-30_直播大屏采集日志.jsonl`

CSV 都带中文表头，可直接用 Excel 打开。表格会包含 `主播`、`商品点击率`、`商品转化率`，主播按你填写的排班自动匹配。

## 自动停止规则

到直播结束时间后，工具会继续在下一个 5 分钟整点检查一次。如果这 5 分钟内累计数据没有变化，就自动停止，并且不写入重复行。

## 本地运行

```bash
npm install
npm run dev
```

## 打包 Windows 安装包

```bash
npm run pack:win
```

打包产物在 `release/` 文件夹。
